INSTALL:

```bash
$ git clone https://github.com/fuhaiwei/springboot_security_restful_api.git
```

RUN SERVER:
```bash
$ cd springboot_security_restful_api
$ mvn clean compile spring-boot:run
```

RUN UI:
```bash
$ cd react-app
$ yarn start
```

Open the following page: http://localhost:3000
